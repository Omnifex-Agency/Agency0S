## 3. The "Obsidian" Design System

We completely refactored the design system to achieve a "Linear-like" premium feel.

### 3.1 Color Palette
*   **Backgrounds**: Deep Slate (`#0f172a` - Slate 950) for infinite depth.
*   **Accents**: Electric Indigo (`#6366f1`) for primary actions and focus rings.
*   **Success**: Teal (`#14b8a6`) for growth metrics and positive states.
*   **Glassmorphism**: Heavy use of `backdrop-blur-xl`, `bg-slate-900/50`, and `border-white/10` to create hierarchy without heavy shadows.

### 3.2 Key UI Components
*   **Collapsible Sidebar**: A fully animated navigation dock. It expands/collapses with a spring physics animation (`stiffness: 200`) and features a glowing active-tab indicator.
*   **Bento Grid Dashboard**: The main view (`/app/page.tsx`) uses a CSS Grid layout to organize key metrics, charts, and activity feeds into a clean, cohesive "Bento Box" structure.
*   **Micro-Interactions**:
    *   **Inputs**: Glow with an Indigo ring upon focus.
    *   **Cards**: entry animations (`opacity` + `y-axis` slide) on page load.
    *   **Links**: Subtle background fade-ins on hover.
